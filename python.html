¿Por qué me obsesioné?

Todo comenzó el 16 de enero del 2020 cuando, en medio de un asado -cocinado por mi, obviamente- y, "solo queriendo ser popular" como el resto, dije que también queria participar en el taller que Django Girls estaba organizando para el 6 de febrero.

Me tocó comprar un computador portatil de rapidez, porque tenia uno de mesa. La experiencia fue bastante interesante, lo primero que aprendi fue a usar GitHub antes de Django jaja. Despues de ese día comencé a buscar bootcamps de Python y encontre a Acamica pero en la carrera de Data Science, para la cual pedían conocimiento en Python y yo no tenia ninguno.

Como tenia un mes hasta la fecha limite de pago de la inscripción, tome como reto aprender a programar basicamente en Python por mi cuenta. Conseguí un libro que a nadie le gusto sobre Python2 orientado a objetos y decidí pasarlo a la version 3 para afianzar sintaxis.

Por lo que, despues de ese mes, puedo decir que manejo Python2 y Python3 nativo básico :)
